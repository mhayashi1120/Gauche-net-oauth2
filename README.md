Gauche-net-oauth2
===================

Not like Oauth 1.0, Oauth2 is a simple protocol.
This package desired to provide convenience (utility) procedure to handle Major Oauth2 provider's (Github, Facebook, Google ...) API.

## Install

    ./configure
    make check
    sudo make install

## Confirmed OAuth2 Providers

- Google (2022-12)
- Twitter (2022-12)
- Github
- Facebook
- LINE
